# 2026-05-17 TrendRiderStrategy 部署

## 策略设计总结

TrendRiderStrategy 历时 10+ 轮迭代，最终达到 +16.33%（1个月）回测表现。

**核心设计：**
- 15m 入场 + 4h 趋势过滤（EMA20/50交叉 + ADX）
- 成交量确认（OBV + 量比）
- 动态杠杆 2x-5x（基于4h ADX强度）
- 宽止损(-0.25) + 追踪止损（盈利10%后启动5%追踪）
- 无ROI退出，让利润充分奔跑

**关键发现：**
- freqtrade 会对 stoploss/ROI/trailing_stop 除以杠杆倍率，宽参数 + 高杠杆 = 合理实际止损
- trailing_stop 退出跨3个月保持 100% 胜率
- 添加 exit_signal（4h趋势反转退出）反而增加亏损，不如让追踪止损自然管理

### 回测注意事项

- TrendRiderStrategy 使用5m回测时不会自动下载15m/4h数据，需确保数据目录中存在 `futures/` 下的15m和4h feather文件
- 使用 `config_fott.json`（futures模式）配置进行回测
- 正确回测命令：
  ```
  freqtrade backtesting --config user_data/config_fott.json \
    --strategy TrendRiderStrategy --strategy-path user_data/strategies \
    --datadir user_data/data/binance --userdir user_data --timerange YYYYMMDD-YYYYMMDD
  ```
