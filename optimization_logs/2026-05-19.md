## ⚡ 技术改进

### 杠杆算法升级
```python
# 旧版本：仅基于 ADX 的简单阶梯
if adx > 45: return min(30.0, max_leverage)
elif adx > 40: return min(15.0, max_leverage)
...

# 新版本：多因子动态调整
if adx > 35 and rsi_healthy and trend_strong and volume_confirmed:
    base_leverage = 20.0  # 25.0, 30.0 根据不同策略
else:
    base_leverage = max(3.0, base_leverage * 0.5)  # 风险控制
```

### 风险控制机制
- **基础杠杆**: 所有策略默认 3x
- **动态上限**: 根据策略特点 15x-30x
- **自动降杠杆**: RSI 超买、ADX 过低、趋势反转时自动降低 50%
- **最小杠杆**: 不低于 3x
