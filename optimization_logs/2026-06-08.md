# 2026-06-08/09 — SmallCapHunterV1 穿越信号修复

## 问题诊断

8087 (SmallCapHunterV1) 实盘表现：38笔已平仓，-17.17 USDT，胜率 36.8%，Profit Factor 0.29。

初步怀疑 `pytrendseries detecttrend` 有未来函数，但经分析确认该库有 `limit` 确认机制，不是裸穿越。

**真正根因：入场过滤器太松。**

```python
# 修复前：做多只要"不是看空"就允许 → 方向不明的市场也在交易
long_base = (
    (ott_dir_1d != -1) &   # 0(无趋势) ✅ 1(看涨) ✅ → 太松
    (ott_dir_4h != -1) &
    (ott_dir_15m != -1) &
```

pytrendseries 在趋势不明确时返回 0（无趋势），`!= -1` 允许了所有无趋势期入场，等于在走震荡的币种上反复交易。

## 修复方案：MTF 多周期共识

参考 StrategyChanPy 的 ≥2/3 过滤逻辑：

```python
# 修复后：≥2/3 周期明确看涨/看空才允许
bullish_mtf = ((ott_dir_1d==1) + (ott_dir_4h==1) + (ott_dir_15m==1)) >= 2
bearish_mtf = ((ott_dir_1d==-1) + (ott_dir_4h==-1) + (ott_dir_15m==-1)) >= 2
```

### 改动清单

| 项目 | 修复前 | 修复后 |
|------|--------|--------|
| 入场过滤器 | `!= -1` (方向不明也入场) | MTF 共识 ≥ 2/3 |
| DCA 加仓次数 | 3次 | 2次 (max_entry_position_adjustment=1) |
| 杠杆 | 固定 2-5x | 动态 2-5x (MTF=3→5x, 2→3x, 1→2x) |
| trend fallback | fillna(1) → 偏多 | fillna(0) → 中性 |
| BTC 默认 | 1 (阻断做空) | 0 (允许做空) |
| pytrendseries | 保留 | 保留 (非未来函数) |

## 2026-06-09 — 动态止损与杠杆天花板探索

### 目标
应要求尝试提高杠杆（30x/15x/5x）+ 动态止损。

### 实验过程

| 杠杆配置 | 止损方式 | 结果 | 原因 |
|---------|---------|------|------|
| 30x/15x/5x | custom_stoploss (ATR) | -981 USDT, 0% WR | 30x清算3.3%, ATR就1-3%, 无止损空间 |
| 15x/10x/5x | custom_stoploss (ATR*2) | -981 USDT, 0.4% WR | 15x清算6.7%, 仍太紧 |
| 15x/10x/5x | tiered固定 -5%/-7%/-12% | -982 USDT, 2.5% WR | 15x止损太紧 |
| 15x/10x/5x | 固定 -12.5% | -980 USDT, 3.8% WR | 15x清算<12.5%, 止损在清算线下 |
| 8x/5x/3x | 固定 -12.5% | -420 USDT, 33.8% WR | 8x清算12.5%=止损, 零缓冲 |
| 7x/4x/2x | 固定 -12.5% | -220 USDT, 38.3% WR | 7x清算14.3%, 缓冲1.8%太紧 |
| 6x/4x/2x | 固定 -10% | -212 USDT, 35.4% WR | 止损改紧后反而更差 |
| 5x/3x/2x | 固定 -10% | -34 USDT, 41.3% WR | 止损过紧 |
| **5x/3x/2x** | **固定 -12.5%** | **+24 USDT, 46.2% WR, 4.9% DD** | **唯一稳定盈利** |

### 结论：3m小币种杠杆天花板

3m 周期小币种的 ATR 通常在 1-3% 之间。杠杆上限由以下公式决定：

```
最大杠杆 < 1 / (止损比例 + ATR缓冲 + 费率)
        < 1 / (0.125 + 0.02 + 0.008)
        < 1 / 0.153
        < 6.5x
```

| 杠杆 | 清算距离 | 可行止损范围 | 可行性 |
|------|---------|-------------|--------|
| 30x | 3.3% | 无 | 不可行 |
| 15x | 6.7% | 无 | 不可行 |
| 10x | 10% | 几乎无 | 极难 |
| 7x | 14.3% | -10% 可行但紧 | 勉强 |
| **5x** | **20%** | **-12.5% 安全** | **可行** |

**只有5x及以下才能在3m小币种上同时拥有有效的止损和不被清算。**

### 最终部署配置

- 杠杆: 5x(3共识) / 3x(2共识) / 2x(1共识)
- 止损: 固定 -12.5% + 跟踪止盈
- `use_custom_stoploss = False`（固定止损在当前杠杆下已足够安全）
- 保留 `custom_stoploss` 方法代码供未来参考，但不启用
